Links

Op deze pagina kan je links vinden die ik doorheen mijn opleiding gebruikt heb.

  • Colorzilla is een handige tool voor het gebruiken van kleuren in je website. Je kan er hex- en rgbcodes van kleuren op websites mee genereren, alle kleuren op een website analyseren etc. Daarnaast bezit het ook nog een handige tool voor het maken van gradients.
  • W3C: de webiste van het World Wide Web Consortium. Bevat interessante informatie over webstandaarden. De website heeft ook validators waarmee je je code kan controleren.
  • Anysurfer is een project van Blindenzorg Licht en Liefde en bevat richtlijnen om websites toegankelijk te maken voor mensen met een visuele beperking.
  • Via Codecademy kan je op een interactieve manier code leren, zoals Python, Java, HTML...
